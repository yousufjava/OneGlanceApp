## Deployment

This project is configured for deployment on Vercel. To deploy:

1. Install the Vercel CLI:
```bash
npm install -g vercel
```

2. Login to Vercel:
```bash
vercel login
```

3. Deploy the project:
```bash
vercel
```

For production deployment:
```bash
vercel --prod
```
